# Nurtured Choice Sales & Distribution System

Enterprise sales, distribution, inventory, invoicing, and reporting platform for Nurtured Choice Products.

## Phase 1 Deliverables

- System architecture
- Database design
- API surface definition
- Folder structure
- Delivery roadmap

## Core Stack

- Frontend: React 19, TypeScript, Vite, Tailwind CSS, shadcn/ui, React Router, TanStack Query, React Hook Form, Zod, Recharts
- Backend: ASP.NET Core 9 Web API, Entity Framework Core
- Database: PostgreSQL
- Auth: Google OAuth, JWT, refresh tokens, RBAC
- Architecture: Clean Architecture, repository pattern, DTOs, DI, validation, structured logging, global exception handling

## Phase Roadmap

1. Architecture and schema
2. Database and EF Core models
3. Backend API implementation
4. Frontend implementation
5. Integration and auth
6. Reports and analytics
7. Testing and hardening
8. Performance optimization
